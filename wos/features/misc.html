Misc features

This documentation is not finished yet.

Badges

Badges were always so annoying, giving badges to someone if they just touched a checkpoint. Even if they weren't alive, or were exploiting. Those days are far gone. Now, all you have to do is type the stage number, and then the BadgeID. It's really that simple, it takes longer to make the badge.


Redacting Strings (Chat)

Tired of people spamming you for admin? Fret no longer. Easy tag common annoying terms out of existance

Note: This feature is only available with the Legacy/Default chat. Do not use it with TextChatService.


Progress Bar

The progress bar is a simple feature that lets the user knows how close they are to completing an obby. If a user has completed 74 stages out of 100, the progress bar will be 74% to the end of the container.


System Messages

System Messages are a great way to subtly promote players liking your game, or purchasing gamepasses.


Join And Leave

Join And Leave chat messages are a common thing found in many obbies. You can change what the messages say and the color of them. The default message is "@{username} joined the game." As you can guess, "{username}" will automatically change to the player's username. This is only what is available, and if you guessed "{display_name}" would change to the player's display name, that is not correct.


Features are available in 0.8.* unless stated otherwise.
