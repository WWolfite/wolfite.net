Checkpoints

Checkpoints are self-explanatory. This page will go over all the extra features around checkpoints in WOS.


Animations

Checkpoint animations are easy to edit. They happen once a player reaches a new checkpoint. There are currently 5 types of animations you can use. Classic (defualt), Color-only, Classic-no-color, spin, world. Alternatively, you can use "None" if you don't want an animation.


Sound Effect

The sound effect that is played when a player reaches a new checkpoint. If there is no ID in Configurations, no sound effect will play.


Checkpoint Color

The custom checkpoint color (ReachedCheckpointColor) is the color the checkpoint turns after a player reaches it.


Included in version 0.8.
