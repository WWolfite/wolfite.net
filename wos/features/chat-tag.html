About Chat Tags

Chat tags are pretty simple. There's 4 things you can customise; TagText, TagColor, NameColor, and ChatColor. For example, if TagText was "DEV", and the TagColor was white, as well as the NameColor being cyan and the ChatColor being lime, it would look something like this:

[DEV]

Username:

Chat Text


Sound Effect

The sound effect that is played when a player reaches a new checkpoint. If there is no ID in Configurations, no sound effect will play.


Checkpoint Color

The custom checkpoint color (ReachedCheckpointColor) is the color the checkpoint turns after a player reaches it.


Included in version 0.8.
